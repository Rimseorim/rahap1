# CrossFit Rehab Guide — 디자인 스펙
v1.0 | 2026.05.03

---

## 1. 원칙

- 1화면 1질문. 선택 즉시 다음 화면으로 전환 (로딩 없음)
- 진행 바는 모든 화면에서 항상 노출
- 위험 신호 감지 시 색상 + 텍스트 동시 전환, 재활 루트 완전 차단
- 뒤로 가기 상태 유지 (브라우저 뒤로 가기 = 이전 질문 복원)

---

## 2. 레이아웃

- 기준 해상도: 375px (iPhone SE)
- 최대 너비: 480px, 중앙 정렬
- 여백: 좌우 20px, 상하 24px
- 단일 컬럼. 가로 스크롤 없음
- 컴포넌트 간 간격: 16px 단위 (16 / 32 / 48)

## 4. 타이포그래피

폰트: **Pretendard** (Google Fonts CDN 또는 WOFF2 로컬)

| 레벨 | 용도 | Size | Weight |
|------|------|------|--------|
| T1 제목 | 화면 메인 질문 | 22px | 700 |
| T2 부제목 | 섹션 헤더, 운동명 | 17px | 600 |
| T3 본문 | 설명, 이유, 안내 | 15px | 400 |
| T4 캡션 | 보조 라벨, 단계 표기 | 13px | 400 |

- 행간(line-height): T1 1.4 / T2·T3 1.6 / T4 1.5
- 자간: 기본값 그대로 (Pretendard 기본 최적화됨)
- 색상: T1·T2 → `#1A1A2E` / T3 → `#1A1A2E` / T4 → `#6B7280`

## 5. 컴포넌트

### ProgressBar
- 위치: 화면 최상단 고정
- 높이: 6px, 배경 `#E5E7EB`, 채움 `#2B4490`
- 퍼센트 텍스트 우측 표시 (T4, `#6B7280`)

### QuestionCard
- 배경 white, 모서리 12px, 그림자 `0 2px 8px rgba(0,0,0,0.06)`
- T1으로 질문, T3으로 보조 설명 (선택)

### ChoiceButton
- 전체 너비, 높이 56px, 모서리 10px
- 기본: 테두리 `#D1D5DB`, 배경 white, 텍스트 T2 `#1A1A2E`
- 선택됨: 테두리 `#2B4490`, 배경 `#EEF2FF`
- 터치 영역 최소 48px 보장

### StageIndicator
- 단계: 맨몸 → PVC → 빈 바 → 바벨
- 현재 단계 강조: Navy 배경 + white 텍스트 (T4)
- 완료 단계: `#16A34A` 체크 표시

### ExerciseCard
- T2 운동명, T3 "왜 이 운동인가" 설명 (필수)
- 접힘/펼침 없음 — 설명은 항상 노출

### DangerAlert
- 배경 `#FEF2F2`, 테두리 `#DC2626` 2px
- T2 빨간색으로 경고 제목
- T3 병원 권유 사유 + 병원 찾기 버튼

---

## 6. 화면 맵 (플로우)

```
홈
└─ 동작 선택 (백스쿼트 / 프론트스쿼트 / OHS)
   └─ 통증 부위 선택
      └─ 질문 1 → 질문 2 → ... (3~6개)
         ├─ [위험 감지] → DangerAlert → 병원 안내 (종료)
         └─ 원인 확정 (A / B / C)
            └─ 루트 배정 화면
               └─ 단계 뷰 (StageIndicator + ExerciseCard 목록)
                  └─ 완료
```

| complete | `#F5F6FA` | Green | 루트 배정 완료 |
