CrossFit Rehab Guide — 원페이지 기획서
v1.0 | 2026.05.03 | 물리치료사가 설계한 크로스핏 통증 재활 가이드


WHY — 왜 만드는가

크로스피터는 통증이 생기면 두 가지 중 하나를 선택한다: 참고 훈련하거나, 무작정 쉰다.
둘 다 틀렸다. 같은 무릎 통증도 원인이 발목이면 발목을, 둔근이면 둔근을 고쳐야 한다.
물리치료사가 매번 옆에 있을 수 없다. 이 웹이 그 간격을 메운다.


WHO — 누구를 위한가

주 사용자: 2년차 크로스피터, 주 4회, 무릎 통증 2주째
  사용 상황: 박스에서 스마트폰으로 운동 전후 5분
  핵심 니즈: "내 통증 원인이 뭔지, 지금 뭘 해야 하는지"

보조 사용자: 박스 코치, 회원 재활 상담에 근거가 필요함
  사용 상황: 태블릿으로 회원과 함께 보며 안내
  핵심 니즈: "틀리지 않는 근거 있는 설명"


WHAT — 무엇을 만드는가

원인 감별 → 맞춤 재활 루트 배정 웹 서비스

동작 선택 → 통증 부위 선택 → 질문 + 셀프 테스트 → 원인 특정 → 재활 루트 A/B/C → 진도 체크

· v1 커버: 스쿼트 3종 (백스쿼트·프론트스쿼트·오버헤드스쿼트)
· 각 동작 × 통증 부위마다 원인 3개 분기, 단계별 재활 루트 (모빌리티→맨몸→PVC→빈 바→바벨)
· 복합 원인 시 우선순위 규칙 내장 (구조적 원인 → 과부하 순)
· 병원 권유 조건 자동 감지 (빈 캔 테스트 양성 등)


HOW — 어떻게 작동하는가

1. 원인을 먼저 특정한다
   통증 부위를 선택하면 바로 운동을 주지 않는다. 질문 3~6개 + 셀프 테스트로 원인을 좁힌 뒤에야 루트를 배정한다. 같은 무릎 통증이어도 루트가 달라진다.

2. 이유를 함께 설명한다
   모든 운동 처방에는 "왜 이 운동인가"를 붙인다. 운동 목록이 아니라 물리치료사의 설명처럼 읽혀야 한다.

3. 단계를 작게 쪼갠다
   맨몸 → PVC → 빈 바 → 바벨 순서로 부하를 올린다. 사용자가 지금 어디 있는지 항상 보이게 한다.

4. 이탈보다 완주를 설계한다
   1화면 1질문, 진행 바 표시, 뒤로 가기 상태 유지. 박스에서 스마트폰으로 5분 안에 루트 배정까지 완료할 수 있어야 한다.

5. 위험 신호는 즉시 멈춘다
   병원 권유 조건(회전근개 손상 의심 등) 감지 시 재활 루트 대신 병원 안내로 전환한다. 자가 해결의 한계를 서비스가 먼저 말한다.


METRICS — 성공 기준 (출시 3개월)

· 감별 완료율 (동작 선택 → 루트 배정까지): 60% 이상
· 재활 루트 진입율 (1단계 이상 수행): 40% 이상
· 월간 활성 사용자 (MAU): 200명
· 코치 피드백 수집: 3명 이상


RISK — 주요 리스크

· 자가 진단 오류로 부상 악화 → 면책 고지 + 병원 권유 조건 자동 감지
· 외부 영상 링크 깨짐 → 대체 링크 2개 사전 확보, 6개월 주기 점검
· 감별 도중 이탈 → 1화면 1질문, 진행 바 표시, 뒤로 가기 상태 유지
